### FCC China组织者手册 
FCC China经过一年的发展，目前已在30多个城市建立了线下学习小组。我们非常开心看到越来越多的camper加入到组织者队伍中，为当地程序员们打造一个交流、学习、协作的有爱的社区。我们也相信这群志同道合的伙伴可以一起做更多有趣、有意义的事情！Learn to Code and Tech for Good!
